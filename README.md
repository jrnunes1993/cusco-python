# Cusco Python

## Features

- Pin routing using A* search algorithm
- Pseudo layout display in PNG

## Install & Usage

**Clone the repository**:
```bash
git clone https://github.com/jrnunes1993/cusco-python
```
**Install dependencies**: 
```bash
pip3 install
```

**Running**:

```bash
python3 main.pty
```
